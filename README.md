# prpc

A modern Python library for prpc.

## Features

- Built with Pydantic v2.
- Fully asynchronous with AnyIO and HTTPX.
- Beautiful CLI powered by Typer and Rich.

## Installation

For developers:
```bash
uv sync
```

To add to your project:
```bash
uv add prpc
```


## Quick Start

```python
import anyio
from prpc import Client

    # Implementation coming soon
    pass

if __name__ == "__main__":
    anyio.run(main)
```
