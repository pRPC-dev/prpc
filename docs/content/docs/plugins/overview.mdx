# Plugins

pRPC plugins are **extensions** that add behavior or tooling on top of the core protocol and adapters.

## Available Plugins

<Card title="Code Generation" href="/docs/plugins/prpc-codegen">
  Generate TypeScript or Python client SDKs from your pRPC router
</Card>

## Installation

Install plugins alongside the core `prpc` package as needed:

```bash
pip install prpc prpc-codegen
```
